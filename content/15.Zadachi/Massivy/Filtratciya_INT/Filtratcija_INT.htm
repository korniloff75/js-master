Условие

Напишите функцию, принимающую аргументом массив с разными типами элементов, и возвращающую массив со всеми элементами типа INT, содержащимися во входящем массиве.

Решение

1,'2',3,'4.5',4.555,'a','b',0,15

Исходный код javascript

Также, с небольшой доработкой, можно разнообразить исходную задачу, создав функции, фильтрующие массивы по любому типу данных.

Если требуемый тип данных определяется оператором typeof, то в этом случае -- предпочтительнее использовать его для фильтрации.
