Работа этого скрипта очевидна из примера ниже. Скрипт выводит при каждой новой загрузке страницы случайные строки текста, заданные в нем.

К особенностям скрипта можно отнести:

  1. Расширение String.prototype пользовательской функцией, что позволяет использовать её в качестве метода для любого из экземпляров String. Этот способ среди программистов не является best practice, однако однозначно соответствует общей идеологии javascript и имеет свои преимущества.
  2. Выведение текста стилем "печатной машинки" - по букве с интервалом.
  3. С небольшой доработкой скрипт можно использовать для популярных блоков случайный анекдот или случайный афоризм, а также выводить информацию для пользователей сайта в оригинальной манере.

Количество выводимых строк может быть ограничено лишь возможностями броузера пользователя.
