Не все эффекты можно достичь исплючительно CSS стилями. Рассмотрим пример органичного использования сочетания CSS / Javascript.

При присвоении class="bg"изображения поменяют свой фон на градиентный. Это будет полностью выполнено при помощи CSS. Однако, если мы захотим добавить нестандартную тень под изображение, одним CSS мы уже справиться не сможем.

При присвоении class="shad"под каждым изображением появится такая тень. Ссылка для скачивания файла с использованной в примере тенью находится под самим примером ниже. При использовании этого класса уже не имеет значения прозрачность фона основного изображения.

При присвоении мультикласса class="shad bg"описанные выше свойства классов объединяются, создавая объединенный эффект оформления изображений.

В данном примере мы объединили способность скрипта изменять DOM-модель с эффектами CSS .Как видно, таким образом можно создавать очень интересные способы оформления элементов сайта.
